### Arithmetic Operators

Used for mathematical calculations.  
`+`, `-`, `*`, `/`, `%`, `**` (exponentiation), `++`, `--`.

```js
let a = 10, b = 3;
a + b; // 13
a % b; // 1
a ** b; // 1000
a++; // 11
```

---

### Assignment Operators

Used to assign or update variable values.

`=`, `+=`, `-=`, `*=`, `/=`, `%=`, `**=`

```js
let x = 5;
x += 3; // 8
x *= 2; // 16
```

---

### Comparison Operators

Return boolean results for comparisons.

`==` (loose equality)  
`===` (strict equality)  
`!=`, `!==`, `<`, `>`, `<=`, `>=`

```js
5 == "5";  // true (type coercion)
5 === "5"; // false (no coercion)
null == undefined; // true
null === undefined; // false
```

Use `===` and `!==` for predictable results.

---

### Logical Operators

`&&` (AND), `||` (OR), `!` (NOT)

- Short-circuits: stops evaluation once the result is known.
    
- Returns actual operand, not necessarily boolean.
    

```js
true && "Hello"; // "Hello"
false || "Default"; // "Default"
!0; // true
```

---

### Bitwise Operators

Operate on binary representations of numbers.

`&` (AND), `|` (OR), `^` (XOR), `~` (NOT), `<<` (left shift), `>>` (right shift), `>>>` (unsigned right shift)

```js
5 & 1; // 1 (0101 & 0001)
5 | 2; // 7 (0101 | 0010)
~5;    // -6 (bitwise NOT)
```

Used in low-level optimizations, masks, and permissions systems.

---

### Nullish Coalescing (`??`)

Returns the right-hand operand only if the left is `null` or `undefined`.  
Unlike `||`, it doesnâ€™t treat falsy values like `0` or `""` as missing.

```js
let count = 0;
let result = count ?? 10; // 0
let name = null ?? "Anonymous"; // "Anonymous"
```

---

### Optional Chaining (`?.`)

Safely access nested properties without throwing errors if something is `null` or `undefined`.

```js
const user = { profile: { name: "Gautam" } };
console.log(user.profile?.name); // "Gautam"
console.log(user.settings?.theme); // undefined (no error)
```

---

### Ternary Operator

Short-hand for simple conditional logic:  
`condition ? exprIfTrue : exprIfFalse`

```js
let age = 20;
let msg = age >= 18 ? "Adult" : "Minor";
```

---

### Spread (`...`) Operator

Expands elements of an array or object.

```js
let arr = [1, 2, 3];
let copy = [...arr]; // [1, 2, 3]

let obj = { a: 1, b: 2 };
let newObj = { ...obj, c: 3 }; // { a: 1, b: 2, c: 3 }
```

---

### Rest (`...`) Operator

Collects remaining elements into an array or object.  
Used in function parameters or destructuring.

```js
function sum(...nums) {
  return nums.reduce((a, b) => a + b);
}
sum(1, 2, 3); // 6

const [first, ...rest] = [1, 2, 3, 4];
// first = 1, rest = [2, 3, 4]
```

---

### Destructuring

Extract values from arrays or objects into variables.

#### Array Destructuring

```js
const [a, b, c] = [10, 20, 30];
const [x, , y] = [1, 2, 3]; // skip values
```

#### Object Destructuring

```js
const user = { name: "Gautam", age: 21 };
const { name, age } = user;
```

#### Nested Destructuring

```js
const data = { person: { info: { name: "Gautam" } } };
const { person: { info: { name } } } = data;
```

You can also use default values:

```js
const [p = 5] = [];
const { city = "Unknown" } = {};
```
