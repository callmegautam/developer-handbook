
### `var`, `let`, `const` — Scope and Hoisting

- `var` is **function-scoped**, hoisted and initialized as `undefined`. Redeclaration allowed.
    
- `let` and `const` are **block-scoped**, hoisted but **not initialized** (Temporal Dead Zone).
    
- `const` must be initialized at declaration and cannot be reassigned, but object and array contents can still mutate.
    

```js
console.log(a); // undefined
var a = 5;

console.log(b); // ReferenceError (TDZ)
let b = 10;

const obj = { name: "JS" };
obj.name = "ECMAScript"; // allowed
```

### Primitive Types

Immutable, stored directly in memory, and compared by **value**.

1. **String** → `"hello"`, `'world'`, `` `template` ``
    
2. **Number** → `42`, `3.14`, `NaN`, `Infinity`
    
3. **Boolean** → `true`, `false`
    
4. **Undefined** → declared but not assigned
    
5. **Null** → intentional absence of value
    
6. **Symbol** → unique identifiers (`Symbol('id')`)
    
7. **BigInt** → large integers beyond `Number.MAX_SAFE_INTEGER` (`123n`)
    

```js
let id = Symbol('id');
let big = 123456789012345678901234567890n;
```

---

### Non-Primitive Types

Mutable and compared by **reference**.

- **Object:** key–value pairs
    
    ```js
    const person = { name: "Gautam", age: 21 };
    ```
    
- **Array:** ordered collection
    
    ```js
    const nums = [1, 2, 3];
    ```
    
- **Function:** first-class object
    
    ```js
    function greet() { console.log("Hi"); }
    ```
    

```js
let obj1 = { a: 1 };
let obj2 = { a: 1 };
console.log(obj1 === obj2); // false (different memory references)
```

### Type Checking

- `typeof` → returns type name as string
    
- `instanceof` → checks prototype chain
    
- `Array.isArray()` → specifically detects arrays
    

```js
typeof null; // "object" (bug in spec)
typeof []; // "object"
Array.isArray([]); // true
console.log([] instanceof Array); // true
```

---

### Type Conversion vs Type Coercion

**Type Conversion:** Explicit conversion (manual).  
**Type Coercion:** Implicit conversion (automatic by JS).

```js
// Conversion
Number("10"); // 10
String(123);  // "123"
Boolean(0);   // false

// Coercion
"5" * 2; // 10 (string → number)
"5" + 2; // "52" (number → string)
```

### Truthy and Falsy Values

**Falsy values:**  
`false`, `0`, `-0`, `""`, `null`, `undefined`, `NaN`

Everything else is **truthy**, including `"0"`, `"false"`, `[]`, `{}`, and functions.

```js
if ("0") console.log("truthy"); // executes
if (0) console.log("falsy"); // skipped
```
